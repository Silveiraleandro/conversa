How can I create an event?

Firstly the user needs to sign up and create an account. Shortly after doing that the userwill have full access to the application. The user will be taken to a user page where there is a option to create conversa events.

Como funciona um conversa event?

An user creates a conversa event setting up a day, time and location. The conversa event will also have a description of how many participants are expected, the language level and the subject / topic, which will be discussed. So if you are an user you can create an conversa event in these therms or join an existing event. To do so, you need to click in the button join conversa event and attend it in the day.

Do I need to pay for anything?

Absolutely not! CONVERSA is 100% free and is here to help the massive language lovers community around the world to get together and practice their speaking skills.

How do I report a technical problem?

The Conversa team is always ready to provide the best service as possible to users. Our team appreciates any suggestions or reports to errors in our system. To communicate with us users need to go the contact us page and place their complaints.
